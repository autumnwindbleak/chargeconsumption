###################################  How to run    ###############################################
Exercise 1:
Step 1: get into folder: GuiOptimiser and compile Calculation.java under calculation package.
	type these code in terminal to compile: javac Calculation.java 
Step_2: run Calculation.java
	type these code in terminal to run: java Calculation

The result will show in the terminal and the pic we use is under input folder

Exercise 3:

Step 1: get into folder: MOEAFramework-2.12 and compile main.java under folder src.
	type these code in terminal to compile: javac main.java 
Step_2: run main.java with arguments:[algorithm, problem, populationsize] (we only solve problem ZDT1 and ZDT2, don't put other problem)
	type these code in terminal to run for example: java main 20 NSGAII ZDT1
